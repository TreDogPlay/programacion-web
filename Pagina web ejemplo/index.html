Contáctanos a traves de este formulario

Documentos digitales

Documentos totalmente digitalizados, siempre disponibles para su edición durante el proceso de planeación. Descargue las planeaciones para su firmado

Administración de usuarios

Gestione los diferentes tipos de usurios y roles al sistema, mediante la asignación de priviegios. Asegure un acceso con su cuenta institucional de Gmail.

Configuraciones

Gestiones las diferentes configuraciones del sistema, tales como notificaciones, control de versiones envios de correo entre otros.
